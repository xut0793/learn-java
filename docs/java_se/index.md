# Java SE

Java SE（Java Standard Edition）是 Java 技术的基础，提供了开发和部署桌面、服务器以及嵌入式环境中 Java 应用程序的核心功能。程序可直接在 JVM 上运行，无需额外服务器支持 ‌。
